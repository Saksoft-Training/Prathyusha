About Movie App

This is a practice Movie Management App built using Angular.

Features include:

  • Routing with multiple pages (Home, Movies, Add Movie, About)
  • CRUD operations on movie data
  • Services with different Dependency Injection methods
  • Two-way data binding using ngModel
  • Structural directives like *ngIf and *ngFor
  • Component communication (Parent → Child and Child → Parent)
  • Responsive and user-friendly UI design
  • Form validation for adding movies
  • Dynamic movie detail display with images, genres, cast, and rating

This app is perfect for practicing Angular fundamentals while building a visually appealing, interactive application.
